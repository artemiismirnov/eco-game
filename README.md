## 🎯 О проекте

"Юный эколог" - это увлекательная многопользовательская браузерная игра, в которой игроки становятся защитниками природы. Пройдите маршрут по городам России, решайте экологические проблемы и помогайте природе вместе с друзьями!

## ✨ Особенности

- 🌍 **Реалистичная карта** с городами России
- 🎲 **Интерактивные задания** на экологическую тематику
- 👥 **Многопользовательский режим** до 6 игроков
- 🏗️ **Система строительства** экологических объектов
- 📈 **Прогресс уровней** и система достижений
- 💬 **Встроенный чат** для общения между игроками
